# Sales Data Analysis using Superstore Dataset

## Synent Technologies – Data Science Internship  
### Intermediate Level – Task 5: Sales Data Analysis

This project is developed as part of the Synent Technologies Data Science Internship Program. The objective of this task is to analyze business sales performance using the Superstore Sales Dataset and generate meaningful business insights. :contentReference[oaicite:0]{index=0}

---

# Project Objective

The goal of this project is to:

- Analyze monthly revenue trends
- Identify top-selling products
- Perform profit analysis
- Generate business insights using data visualization

This project demonstrates how data analytics can help businesses make better strategic decisions.

---

# Dataset Information

### Dataset Used
Superstore Sales Dataset

### Dataset Source
https://www.kaggle.com/datasets/vivek468/superstore-dataset-final

### Dataset Features

| Feature | Description |
|---|---|
| Order ID | Unique order identifier |
| Order Date | Date of purchase |
| Ship Mode | Shipping method |
| Customer Name | Customer details |
| Segment | Customer segment |
| Country | Customer country |
| City | Customer city |
| Category | Product category |
| Sub-Category | Product subcategory |
| Product Name | Product details |
| Sales | Revenue generated |
| Quantity | Quantity sold |
| Discount | Applied discount |
| Profit | Profit earned |
